# PRÁCTICA 2 - API REST Carrito
## Endpoints
Método + Ruta - Descripción
--------------------------------------------
POST /api/carrito - Crear un carrito  
GET /api/carrito - Leer lista de carritos  
GET /api/carrito/1 - Leer el carrito 1  
PUT /api/carrito/1 - Modificar el carrito 1  
DELETE /api/carrito/1 - Borrar el carrito 1  
--------------------------------------------
